## 🚀 Project Overview

**Snap2Shop Pro** is an AI-powered visual shopping assistant that allows users to upload or capture an image and instantly detect multiple products within it.

Using **Google Gemini Vision API**, the app identifies objects, understands their style, and provides smart shopping links along with buying suggestions.

This project combines **computer vision + AI + e-commerce** into a single interactive platform.

---

## 🎯 Key Features

### 🔍 Multi-Object Detection

* Detects multiple purchasable items from a single image
* Works with real-world scenes (rooms, outfits, setups)

### 🧠 AI Style Understanding

* Identifies product style (modern, casual, minimal, etc.)
* Enhances search relevance

### 🛒 Smart Shopping Links

* Direct links generated for:

  * Google Shopping
  * Amazon
  * Flipkart / Myntra

### 💡 AI Buying Suggestions

* Each detected item includes:

  * Buying advice
  * Optimized keywords
  * Better search recommendations

### 📷 Dual Input Modes

* Upload Image
* Real-time Camera Capture

### ⚡ Fast Processing

* Powered by **Gemini 2.5 Flash API**
* Optimized request handling with retry logic

## 🖥️ How It Works

1. Upload or capture an image
2. AI scans and detects all products
3. Each product is analyzed for:

   * Name
   * Style
   * Keywords
   * Buying advice
4. Shopping links are generated instantly

## ▶️ How to Run

### 1. Clone the Repository

```
git clone https://github.com/your-username/snap2shop-pro.git
cd snap2shop-pro
```

### 2. Install Dependencies

```
pip install -r requirements.txt
```

### 3. Run the App

```
streamlit run main.py
```
